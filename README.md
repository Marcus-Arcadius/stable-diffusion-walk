# Stable Diffusion Walker
A stable diffusion walker with Docker environment

### Build Docker

```bash
docker$: make build
```

### Walk
```bash
docker$: ./walk.sh
```

### Generate Video

```bash
docker$: docker pull linuxserver/ffmpeg:latest
docker$: ./video.sh
```
